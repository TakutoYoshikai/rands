# rands
The random string generator.

### Requirements
* macOS or Ubuntu
* npm

### Usage
**install**
```bash
npm install -g TakutoYoshikai/rands
```

**generate random string**
```bash
rands <LENGTH>

# specify characters
rands <LENGTH> <CHARACTERS>
# example 
rands 10 abcd
# cacadbabdb
```

**generate random hex string**
```bash
rands hex <LENGTH>
```
